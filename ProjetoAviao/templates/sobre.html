Sobre o Dataset

Este dataset foi coletado com a ferramenta de web scraping Octoparse a partir do site Ease My Trip, cobrindo opções de reservas de voos nas principais cidades metropolitanas da Índia. A coleta de dados durou 50 dias, de 11 de fevereiro a 31 de março de 2022, capturando 300.261 opções distintas de reserva de voos, com informações sobre voos em classe econômica e executiva.

O dataset está disponível para consulta em: Kaggle - Flight Price Prediction

Características do Dataset
  • Airline: Nome da companhia aérea (6 companhias).
  • Flight: Código do voo.
  • Source City: Cidade de origem (6 cidades).
  • Departure Time: Hora de partida, categorizada em 6 intervalos de tempo.
  • Stops: Número de paradas entre as cidades de origem e destino (3 valores possíveis).
  • Arrival Time: Hora de chegada, categorizada em 6 intervalos de tempo.
  • Destination City: Cidade de destino (6 cidades).
  • Class: Classe do assento (Econômica ou Executiva).
  • Duration: Duração total da viagem entre as cidades (em horas).
  • Days Left: Dias restantes até a data de partida.
  • Price: Preço do bilhete (variável alvo).

Objetivo do Projeto

Este projeto tem dois objetivos principais:

Análise de Dados

A análise de dados busca responder às seguintes perguntas de pesquisa através de estatísticas:

  • O preço varia de acordo com as companhias aéreas?
  • Como o preço é afetado quando os ingressos são comprados com apenas 1 ou 2 dias de antecedência?
  • O preço do bilhete muda com base no horário de partida e chegada?
  • Como o preço varia entre diferentes cidades de origem e destino?
  • Como o preço do bilhete varia entre as classes Econômica e Executiva?
Modelo de Machine Learning

A segunda parte do projeto é a criação de um modelo de Machine Learning. Este modelo usará algumas das características do dataset fornecidas pelo usuário e calculará o preço médio do bilhete com base nas entradas.
